1️⃣ DATE_ADD()
📖 Definition

DATE_ADD() is used to add an interval (days, months, years, etc.) to a date.

📝 Syntax
DATE_ADD(date, INTERVAL value DAY)
💻 Example 1 – Add 10 Days
SELECT
order_id,
customer_name,
order_date,
DATE_ADD(order_date,INTERVAL 10 DAY) AS expected_delivery
FROM Orders;
📊 Output
Order Date	Expected Delivery
2025-01-10	2025-01-20
2025-02-18	2025-02-28
🧠 Dry Run
Order Date

2025-01-10

↓

Add 10 Days

↓

2025-01-20
💻 Example 2 – Add 2 Months
SELECT
customer_name,
DATE_ADD(order_date,INTERVAL 2 MONTH)
FROM Orders;
2️⃣ LAST_DAY()
📖 Definition

Returns the last date of the month for a given date.

📝 Syntax
LAST_DAY(order_date)
💻 Example
SELECT
customer_name,
order_date,
LAST_DAY(order_date) AS month_end
FROM Orders;
📊 Output
Order Date	Month End
2025-02-18	2025-02-28
2025-04-20	2025-04-30
🧠 Dry Run
Order Date

2025-04-20

↓

LAST_DAY()

↓

2025-04-30
🚨 Today's Doubt
❌ Wrong Query
SELECT customer_name
FROM Orders
ORDER BY LAST_DAY(order_date)
LIMIT 1;
❓ Why is it Wrong?

Suppose we have:

Customer	Order Date
Sai	2025-01-10
Ravi	2025-01-20

Both rows become

LAST_DAY()

↓

2025-01-31

Both values are the same.

LAST_DAY() tells SQL when the month ends, not which order came first.

✅ Correct Solution
SELECT
customer_name,
order_date
FROM Orders o
WHERE order_date=
(
SELECT MIN(order_date)
FROM Orders
WHERE YEAR(order_date)=YEAR(o.order_date)
AND MONTH(order_date)=MONTH(o.order_date)
);

3️⃣ DATE_FORMAT()
📖 Definition

Displays dates in different formats.

📝 Syntax
DATE_FORMAT(date,'%format')
Example 1
SELECT
DATE_FORMAT(order_date,'%d-%b-%Y')
FROM Orders;

Output

10-Jan-2025
Example 2
SELECT
DATE_FORMAT(order_date,'%M %Y')
FROM Orders;

Output

January 2025

February 2025

March 2025
📋 DATE_FORMAT Cheat Sheet
Format	Output
%d	01-31
%m	01-12
%b	Jan
%M	January
%y	25
%Y	2025
%W	Monday
%a	Mon
4️⃣ EXTRACT()
📖 Definition

Extracts a specific part from a date.

Extract Month
SELECT
EXTRACT(MONTH FROM order_date)
FROM Orders;
Extract Year
SELECT
EXTRACT(YEAR FROM order_date)
FROM Orders;
Extract Day
SELECT
EXTRACT(DAY FROM order_date)
FROM Orders;
⭐ Interview Questions
Question 1

Find orders placed in the current month.

SELECT *
FROM Orders
WHERE MONTH(order_date)=MONTH(CURDATE())
AND YEAR(order_date)=YEAR(CURDATE());
Question 2

Find orders placed in the previous month.

SELECT *
FROM Orders
WHERE MONTH(order_date)=MONTH(DATE_SUB(CURDATE(),INTERVAL 1 MONTH))
AND YEAR(order_date)=YEAR(DATE_SUB(CURDATE(),INTERVAL 1 MONTH));
Question 3

Find orders placed in the current year.

SELECT *
FROM Orders
WHERE YEAR(order_date)=YEAR(CURDATE());

🎯 Key Takeaways
DATE_ADD() is used to add intervals to dates.
LAST_DAY() returns the last calendar day of a month but cannot identify the first order of a month.
DATE_FORMAT() is used to present dates in user-friendly formats.
EXTRACT() retrieves specific date components such as year, month, or day.
For "first order in every month", use MIN(order_date) with a correlated subquery or ROW_NUMBER() (MySQL 8+), not LAST_DAY().
